# MC-Dropout
Курсовая работа (9 вар.), Хабаров М.А. (20421)

# Описание
Для оценки неопределённости модели в задаче регрессии применяется техника Дропаут МК. Её суть заключается в том, чтобы сохранять слой дропаут активным на этапе предсказания (инференса).

# Запуск
0. Используется версия ```Python 3.10.7```
1. Установка зависимостей: ```pip install -r req.txt```
2. Исполняемым файлом является ```solution.ipynb```

# Структура исп. файла
1. Загрузка и предобработка данных
2. Обучение MLP
3. Использование Дропаут МК на инференсе
4. Получение распределения предсказаний
5. Построение графиков и доверительных интервалов
